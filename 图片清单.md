## 📝 快速添加步骤

### 方法1：手动添加（推荐）
1. 打开 `portfolio/assets/` 文件夹
2. 按照上面的清单，把对应图片复制到相应位置
3. 确保文件名完全一致（包括大小写和扩展名）
4. 刷新浏览器查看效果

### 方法2：从现有文件夹复制
你提到图片在这些位置：
- 个人照片：在 `assets` 文件夹
- YanCare截图：在 `assets` 文件夹的小程序相关位置
- Podcast截图：在 `podcast` 文件夹

可以用以下命令批量复制（根据实际路径调整）：

```bash
# 示例：从原文件夹复制到网站文件夹
cp /原路径/个人简历证件照.jpg portfolio/assets/
cp /原路径/qrcode.jpg portfolio/assets/
# ... 其他文件类似
```

## ⚠️ 注意事项

1. **文件格式**：推荐使用 `.jpg` 或 `.png` 格式
2. **文件大小**：建议每张图片不超过500KB（压缩后网页加载更快）
3. **命名规范**：文件名必须与清单中完全一致
4. **预览测试**：添加图片后记得在浏览器中刷新查看效果
